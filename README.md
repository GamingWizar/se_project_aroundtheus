# Project 3: Around The U.S.

This is the third project of the Web Development program at Triple Ten. It was created using HTML and CSS, based on the design provided through Figma.

## Project features

- Semantic HTML5
- Flexbox
- Grid
- Responsive Layout
- Desktop and Mobile layouts
- Flat BEM
- CSS transform
- Smooth animations
- JS Classes
- Webpack bundling
- API Integration
